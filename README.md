# ALURAMIDI
Projeto criando páginas dinâmicas com JavaScript realizado através dos cursos da ALURA.

# Deescrição
Este projeto consiste em uma bateria virtual onde você bote tocar nos botões e reproduzir os sons de instrumentos.
